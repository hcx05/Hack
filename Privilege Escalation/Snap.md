Snap manage system is aim to resolve software installation problems in different Linux version.

Snap executes in a Sandbox, not nature hosts.

`snap-confine` is a tool that enable Sandbox to communicate with the system, it must have `SUID` to construct an environment for Sandbox.

```bash
/snap/core18/1944/bin/ping
```

core18 : Basic execution environment for Snap.

1944 : version number.
