A database is controlled by a DBMS, which is an acronym for  Database Management System. DBMSs fall into two camps: [[Relational and Non-Relational Databases]].

Columns : Each column, better referred to as a field, has a unique name per table. When creating a column, you also set the type of data it will contain, common ones being integers (numbers), strings (standard text) or dates.

Rows : Rows or records contain individual lines of data. When you add data to the table, a new row/record is created; when you delete data, a row/record is removed.
